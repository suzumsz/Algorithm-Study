# 문자열 다루기 기본 
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-06-06

### Sol
count를 이용해서 4인경우와 6인경우 실행되도록 하고 if문이 실행되면 `filter`와 `isNumber`를 이용해 변수에 숫자만 집어넣는다.
```Swift
import Foundation

func solution(_ s:String) -> Bool {

    if s.count == 4 || s.count == 6{
        let num = s.filter{($0.isNumber)}
        if s.count == num.count{
            return true
        }
    }

    return false
}
```

### Other's Sol
짧고 간결하게 풀 수 있었다.
```Swift
func solution(_ s:String) -> Bool {
    return (Int(s) != nil && (s.count == 4 || s.count == 6)) ? true : false
}
```
