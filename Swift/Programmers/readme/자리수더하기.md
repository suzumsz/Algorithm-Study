# 자리수 더하기
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-05-09

### Sol
자연수 n을 배열로 바꿔 나열해야 하기 때문에, [123]이 아니라 [1,2,3] 형태로 반복문을 사용하여 만들어  주었다.   
그리고나서 `.map` 함수를 사용해서 정수 배열로 바꾸고, String에서 다시 정수형으로 바꾸어 주었다.  
`for in`반복문으로 나열하면서 total 변수에 순서대로 더해 최종 값을 얻었다.
```Swift
import Foundation

func solution(_ n:Int) -> Int
{
    let array = String(n).map {Int(String($0))!}
    var total = 0

    for i in array {
        total += i
    }

    return total
}
```

### Other's Sol
`reduce`를 이용하여 간단하게 풀 수 있는 문제다.
```Swift
import Foundation

func solution(_ n:Int) -> Int
{
    return String(n).reduce(0, {$0+Int(String($1))!});
}
```
