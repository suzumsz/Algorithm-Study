# 평균 구하기
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-05-02

### Sol
`reduce(into:)`를 사용했다.   
`into:`인자에 `0` 초기값을 설정하고 배열의 모든 숫자와 누적합을 저장한다.  
`arr` 크기로 나누어 평균값을 반환한다. 
```Swift
func solution(_ arr:[Int]) -> Double {
    return Double(arr.reduce(into:0) {$0 += $1}) / Double(arr.count)
}
```

### Other's Sol
`reduce`를 다른 형태로 더욱 간결하게 사용할 수 있었다.
```Swift
func solution(_ arr:[Int]) -> Double {

    return Double(arr.reduce(0,+))/Double(arr.count)
}
```
