# 서울에서 김서방 찾기
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-06-06

### Sol
배열 중에 Kim이 있다면 x를 i로 바꾸어주고 '김서방은 x에 있다'를 반환해서 풀 수 있었다.
```Swift
func solution(_ seoul:[String]) -> String {
    var x = Int()
    for i in 0..<seoul.count {
        if seoul[i] == "Kim" {
           x = i
        break
        }
    }
    return "김서방은 \(x)에 있다"
}
```

### Other's Sol
for문을 사용하지 않고 `index(of:)`를 사용하면 해당 index가 출력되도록 할 수 있다.
```Swift
func solution(_ seoul:[String]) -> String {
    return "김서방은 \(seoul.index(of: "Kim")!)에 있다"
}
```
