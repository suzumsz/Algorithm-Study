# 나누어 떨어지는 숫자 배열
> 난이도 : Lv1   
> 유형 : 연습문제
> 2021-04-11

### Sol
쉽게 풀 수 있는 문제였다.  
divisor로 나누었을 때 나누어 떨어지면 `append`로 answer에 넣어준다.  
`isEmpty`로 문자열 안에 값이 비어있는 확인하고 그렇다면 `-1`을 출력하고,  
`sorted`로 정렬해준다.

```Swift
func solution(_ arr:[Int], _ divisor:Int) -> [Int] {
    var answer:[Int] = []
    
    for i in arr {
        if i % divisor == 0 {
            answer.append(i)
        }
    }
    if answer.isEmpty {
        answer.append(-1)
    }
    return answer.sorted()
}
```

### Other's Sol
정렬을 한 후 `filter` 함수를 써서 풀 수 있는 간단한 문제였다.
```Swift
func solution(_ arr:[Int], _ divisor:Int) -> [Int] {
    let array = arr.sorted().filter{ $0 % divisor == 0 }
    return  array == [] ? [-1] : array
}
```
