# 핸드폰 번호 가리기
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-04-18

### Sol
마지막 번호 4자리를 제외한 숫자들은 *로 만들어야 하므로 for문을 사용하여 더해준다.  
마지막 번호 4자리는 `suffix`를 통해 보여준다.
```Swift
func solution(_ phone_number:String) -> String {
    var answer = String()

    for _ in 0..<phone_number.count-4{
        answer += "*"
    }
    answer += phone_number.suffix(4)
    return answer
}

```

### Other's Sol
`String(repeating:,count)`를 이용하여 자동으로 반복되게 할 수 있었다.
```Swift
func solution(_ phone_number:String) -> String {
    return String("\(String(repeating: "*", count: phone_number.count - 4))\(phone_number.suffix(4))")
}
```
