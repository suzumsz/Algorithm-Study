# 약수의 합
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-05-02

### Sol
 
```Swift
func solution(_ n:Int) -> Int {
    var answer = 0
    guard n > 0 else {
        return 0
    }

    for i in 1...n {
        if n%i == 0 {
            answer += i
        }
    }
    return answer
}
```
