# 약수의 합
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-05-02

### Sol
for문을 사용하여 0으로 나누어 떨어질 때 answer에 추가해 주었다.  
0일 때를 대비하지 않아 실패로 떴다!!  
이럴 때에는 `guard`문을 이용해 0일 경우를 대비해주어야 한다!!
```Swift
func solution(_ n:Int) -> Int {
    var answer = 0
    guard n > 0 else {
        return 0
    }

    for i in 1...n {
        if n%i == 0 {
            answer += i
        }
    }
    return answer
}
```

### Other's Sol
`guard`문을 이용해 0일 경우를 대비하고, `Array`와 `filter`를 이용해 
나누어 떨이지는 것을 넣고 `reduce`를 이용해서 차례로 더하여 풀 수 있는 문제였다!!!!
```Swift
func solution(_ n:Int) -> Int {   
    guard n != 0 else {
        return 0
    }
    return Array(1...n).filter{n % $0 == 0}.reduce(0, +)
}
```
