# 두 정수 사이의 합
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-04-14

### Sol
b가 a보다 크면 a부터 b까지 for문을 사용해 더해준다.  
반대로 a가 b보다 크면 b부터 a까지 for문을 사용해 값을 더해주어, 두 정수 사이의 합을 출력할 수 있다.
```Swift
func solution(_ a:Int, _ b:Int) -> Int64 {
    var sum = 0

    if b >= a {
        for a in a...b {
            sum += a
        }
    }else {
        for b in b...a {
            sum += b
        }
    }
    return Int64(sum)
}
```

### Other's Sol
`Array(a...b)`를 이용해 a부터 b까지의 값을 넣은 후 `reduce`를 이용해 0부터 차례대로 더하여 풀 수 있는 문제였다.
```Swift
func solution(_ a:Int, _ b:Int) -> Int64 {
    return Int64(Array(a > b ? b...a : a...b).reduce(0, +))
  }
```
