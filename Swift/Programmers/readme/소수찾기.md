# 소수 찾
> 난이도 : Lv1   
> 유형 : 연습문제   
> 2021-05-16

### Sol
내가 원래 풀었던 부분에서는 시간초과가 떴다.  
찾아보니 에라토스테네스의 체라는 알고리즘으로 풀 수 있는 문제였다.  
n+1만큼 false로 초기 배열을 셋팅해준 후, for문을 사용해 2부터 false이면 count해준다.  
false를 만나면 그 수는 소수를 말한다. `stride`는 from부터 through까지 by만큼을 말한다.
좀 더 공부하자 으쌰으쌰!!
```Swift
func solution(_ n: Int) -> Int {
    var count = 0
    var array = Array.init(repeating: false, count: n+1)
    
    for i in 2...n {
        if array[i] == false {
            count += 1
            
            for j in stride(from: i, through: n, by: i) {
               array[j] = true
            }
        }
    }
    return count
}
```
