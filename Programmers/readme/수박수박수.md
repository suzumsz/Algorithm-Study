# 수박수박수박수박수박수?
> 난이도 : Lv1   
> 유형 : 연습문제  
> 2021-04-06

### Sol
for문을 사용하여 i를 0~n번까지 반복해준다.  
if문을 사용하여 짝수일 때 '수'를 출력하고 홀수일 때 '박'을 출력해준다.  
반복한 수만큼만 '수박' 이라는 글자가 연속적으로 출력되도록 한다.
```Swift
func solution(_ n:Int) -> String {
    var answer = ""
    for i in 0..<n {
        if i % 2 == 0 {
            answer += "수"
        }else { 
            answer += "박"
        }
    }
    return answer
}
```

### Other's Sol
`String` 함수 중에서 특정 문자를 입력받은 수만큼 반복하여 문자를 생성해주는 
`repeat` 함수를 사용해서 풀 수 있는 문제였다.

```Swift
func solution(_ n:Int) -> String {
    return "\(String(repeating: "수박", count: n / 2))\(n % 2 == 0 ? "" : "수")"
}
```
