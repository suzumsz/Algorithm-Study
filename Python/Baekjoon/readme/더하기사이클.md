### Sol
num을 정수형으로 받는다. 이후 체크를 위해 num값을 check에 넣어준다.   
while문을 사용해 두자리 수를 쪼개어 더해준다. 뒷 자리수 끼리의 덧셈을 위해 new_num에 (num%10)*10 + temp%10을 해준다.  
count로 갯수를 세고 new_num값을 num에 다시 넣어주면서 while문을 반복한다.  
만약 new_num과 check값이 동일하다면 멈추고 count를 출력한다.  
이 문제는 //와 %를 잘 활용한다면 풀 수 있는 문제였다.

```python
num = int(input())
check = num
temp = 0
new_num = 0
count = 0

while True:
  temp = num//10 + num%10
  new_num = (num%10)*10 + temp%10
  count += 1
  num = new_num
  if new_num == check :
    break
print(count)
```
